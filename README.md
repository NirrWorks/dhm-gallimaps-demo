# DHM x GalliMaps Demo

Interactive mock dashboard for DHM and GalliMaps disaster/weather features.

## Demo

Open `index.html` or host the repository with GitHub Pages. The root page redirects to:

`dhm-professional-dashboard/`

## Includes

- GalliMaps basemap mock dashboard
- Live weather and air quality mode
- Rainfall stations and rainfall prediction maps
- Hazard map with flood, landslide, fire, quake, storm and heavy rain layers
- Route weather, trekking weather, flood alert and smart city mock modes
